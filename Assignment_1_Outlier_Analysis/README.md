# Assignment 1: Outlier Analysis

## Overview
This project analyzes a dataset (`Data1.txt`) to:
- Calculate mean, median, variance, and standard deviation
- Detect and classify outliers (Moderate, High, Extreme)
- Visualize the data before and after outlier removal
- Show how removing outliers affects statistics

## File Structure
- `data/` - input dataset
- `src/` - Python scripts
- `results/` - generated plots
- `README.md` - project description
- `requirements.txt` - Python dependencies
